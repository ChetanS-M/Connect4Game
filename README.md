# Connect 4 Game

A simple and interactive Connect 4 game built using HTML, CSS, and JavaScript.

## 🎮 About the Game

Connect 4 is a classic two-player strategy game where players take turns dropping colored discs into a 7-column, 6-row grid. The objective is to be the first player to connect four of your discs horizontally, vertically, or diagonally.

## 🚀 Features

- Two-player gameplay
- Red vs Yellow turns
- Automatic disc placement
- Win detection:
  - Horizontal
  - Vertical
  - Diagonal
- Restart game button
- Responsive and clean user interface
- Pure HTML, CSS, and JavaScript (No frameworks)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Game Rules

1. Players take turns dropping discs into one of the seven columns.
2. The disc falls to the lowest available position in the selected column.
3. The first player to connect four discs in a row wins.
4. Connections can be:
   - Horizontal
   - Vertical
   - Diagonal
